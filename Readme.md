## 坚持背单词，坚持健身，坚持阅读 
最近突然发现各种慢性病如胃病，颈椎病随着自己身体体质变差而到来，我现在才意识到拥有健康体质的重要性，也开始着手增加锻炼来加强自己的体质。作为一个苦逼码农，身魄强健更为重要。希望能有一个计划来逐步锻炼自己，并且对自己的弱项如英语和阅读量进行强化。希望大家可以加入，以此自勉。

## 计划内容

1. 每日进行背单词，20min以上
2. 一周坚持3-4次健身，绕校健身跑 `or` 俯卧撑，引体向上练习
3. 坚持每天看`kindle` 30min以上
